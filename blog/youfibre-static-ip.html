Adding a static IP to your YouFibre connection

YouFibre offer a static IP as a paid add-on. If you're self-hosting websites, running a mail server, or need a reliable endpoint for a VPN or remote access, it's worth it. Dynamic IP addresses from residential ISPs can change, which breaks DNS records and remote access configurations.

How to get one

Contact YouFibre support and request a static IP. At the time of writing it's available as a monthly add-on. Check their current pricing, as it may have changed. The process is straightforward: they provision it on their end and provide you with the IP address and subnet details.

Configuring it in OPNsense

Switch the WAN interface from DHCP to static. Enter the IP address, subnet mask, and gateway provided by YouFibre. For DNS, I use Unbound locally rather than YouFibre's resolvers, so there's nothing to change there.

Everything came up cleanly. No additional configuration was needed on the YouFibre side beyond the provisioning request.

What I use it for

Hosting a few personal projects and websites directly from the network, running my own mail server with correct reverse DNS, and as a stable endpoint for remote access into the home office. The static IP has been solid — no surprise IP changes since the switch.
